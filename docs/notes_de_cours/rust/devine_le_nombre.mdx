# 🎲 Devine le nombre

## 🎯 Objectifs pédagogiques

- Écrire son premier programme en Rust.
- Comprendre les bases de la syntaxe Rust.
- Utiliser Cargo pour gérer un projet Rust.
- Lire et comprendre le tutoriel officiel.

## 📜 Énoncé

Vous allez implémenter un jeu de devinette de nombres en Rust. L'énoncé est
disponible dans le livre [The Rust Programming
Language](https://doc.rust-lang.org/book/ch02-00-guessing-game-tutorial.html).

Pendant que vous faites ce tutoriel, essayez de faire le parallèle avec ce que
vous avez appris en Python. Par exemple :

- Est-ce qu'il y a un équivalent de `uv` en Rust ?
- Comment déclare-t-on une variable en Rust ?
- Comment fonctionnent l'affichage et la saisie en Rust ?
- Est-ce que les structures de contrôle sont similaires ?
- Comment fait-on pour exécuter le programme ?

### Équipe

Vous serez en équipe de trois ou quatre. Chaque membre de l'équipe doit
écrire son propre code, mais vous pouvez vous entraider.
