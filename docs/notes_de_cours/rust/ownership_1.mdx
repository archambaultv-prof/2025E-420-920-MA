# 🦀 Ownership en Rust

## 🎯 Objectifs pédagogiques

- Comprendre le concept d'ownership (propriété) en Rust
- Identifier les erreurs liées au transfert de propriété
- Utiliser les références et le borrowing correctement
- Résoudre les problèmes de compilation liés à l'ownership

## 🔧 Exercice 1 : Transfert de propriété

### Problème à résoudre

Le code suivant ne compile pas. Identifiez et corrigez les erreurs :

```rust
fn main() {
    let message = String::from("Bonjour Rust!");
    
    afficher_message(message);
    println!("Message original: {}", message); // ❌ Erreur ici
    
    let longueur = calculer_longueur(message); // ❌ Erreur ici aussi
    println!("Longueur: {}", longueur);
}

fn afficher_message(msg: String) {
    println!("Affichage: {}", msg);
}

fn calculer_longueur(msg: String) -> usize {
    msg.len()
}
```

### 🤔 Questions de réflexion

1. Pourquoi le code ne compile-t-il pas ?
2. Que se passe-t-il avec la variable `message` après l'appel à `afficher_message()` ?
3. Proposez **deux solutions différentes** pour corriger ce problème.
