# Projet PiLedger

Pour rendre l'apprentissage plus concret, nous allons travailler sur un projet de session
appelé PiLedger. Ce projet est une application de comptabilité en ligne de commande qui
permet de gérer des comptes et des transactions.

## Objectifs

- Développer des compétences en programmation Python.
    - Réutiliser les notions du cours de Programmation 1.
    - Mettre en pratique les concepts du cours de Programmation 2.
- Développer une application en ligne de commande.
- Utiliser des paquets Python pour faciliter le développement.
- Comprendre les concepts de base de la comptabilité. *Puisqu'il s'agit d'un
  cours de programmation, cette partie ne sera pas évaluée directement.*

## Fonctionnalités

Voici les fonctionnalités principales que nous allons implémenter dans PiLedger :

- Gestion des comptes
- Gestion des transactions
- Vérification des soldes
- Importation de transactions
- Rapports financiers

## Évaluations

Les travaux pratiques 1, 2 et 3 seront en lien avec ce projet. Les travaux
pratiques 2 et 3 se font en équipe de deux.
